# ComicsApp
Simple Android App created through [Five](https://five.agency/) Android Internship.

App Features:
+ Fetching top 10 trending episodes
+ Providing selected episode details
+ Searching for specific episodes
+ Pull to refresh

Libraries used:
+ Retrofit
+ Gson
+ Glide
+ RxJava
+ Room
